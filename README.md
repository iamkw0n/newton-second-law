# Newton's Second Law Calculator üçé
An interactive calculator for Newton's Second Law

## Formula
$$
F(\text{N}) = m(\text{kg}) \times a(\text{m/s}^2)
$$

Where:
- **F** is for **Force** ($\text{N}$)
- **m** is for **mass** ($\text{kg}$)
- **a** is for **acceleration** ($\text{m/s}^2$)

## Features
- Calculates force only when both inputs are vaild numbers
- Prevents calculation on invaild or empty input
